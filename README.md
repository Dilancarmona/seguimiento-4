# seguimiento-4

En este proyecto se desarrolló un pequeño videojuego usando una ESP32, una pantalla OLED y un joystick analógico. El jugador controla un personaje que debe evitar obstáculos mientras recoge objetos dentro de la pantalla.

El juego cuenta con diferentes modos, sonidos usando un buzzer y menús básicos para seleccionar opciones. Todo el programa fue realizado en MicroPython y probado tanto en simulación como en montaje físico.

La idea principal del trabajo fue aplicar conceptos vistos en clase relacionados con entradas y salidas digitales, lectura analógica, programación de periféricos y lógica de control.
